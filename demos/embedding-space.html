What you're seeing: Each text input is embedded into a 21-dimensional space through the PHDM (Polyhedral Hamiltonian Defense Manifold). The 21 dimensions come from:
- 6 Sacred Tongue activations (Kor'aelin, Avali, Runethic, Cassisivadan, Umbroth, Draumric)
- 6 tongue interaction cross-terms
- 3 temporal dimensions (context, session, history)
- 3 intent dimensions (valence, arousal, dominance)
- 3 structural dimensions (complexity, coherence, entropy)

The 2D projection uses PCA to show the most informative view. Safe inputs cluster near center. Attacks scatter toward the edges. The separation between clusters IS the security guarantee.

Try it: Type normal questions, then type attacks. Watch them land in different regions. The farther apart they are, the easier the system distinguishes them.
