How it works: Each Sacred Tongue has a 256-token vocabulary — one token per byte value. Text is UTF-8 encoded, then each byte maps to a conlang token. The tongue's phi-weighted cost (1.0 for Kor'aelin up to 11.09 for Draumric) determines how expensive governance operations are in that domain. Higher-privilege tongues cost exponentially more, making unauthorized access mathematically infeasible.

Try it: Switch between tongues and watch the same text encode differently. Copy tokens from the output, paste them in the decode box — they decode back perfectly.
