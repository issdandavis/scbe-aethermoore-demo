1. Save the receiptKeep the purchase email. It tells support what was bought and which email address was used.
2. Open the matching manualUse the package page instead of guessing from repo docs or theory pages.
3. Confirm contentsCheck the bundle against the package matrix below.
4. Run one success checkEach package should get to one specific working result first.
5. Expand laterDo not scale up until the first small use works cleanly.
