Delivery path

1. BuyComplete checkout through the package payment link.
2. ConfirmKeep the receipt email and the purchase email address you used.
3. ReceiveLook for the bundle link, manual link, and any key or access note if the product needs one.
4. UseOpen the package manual and follow the first-run checklist for that specific product.
5. VerifyRun the success check at the end of the manual before using the package in real work.
