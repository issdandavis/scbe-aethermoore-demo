HYDRA Agent Templates

This package is for operators who want governed swarm structure without inventing every role and packet from scratch. Use it to launch small coordinated agent groups with clearer ownership and cleaner handoffs.

Agent roles Packet patterns Launch checklists Swarm-ready

First-run path

1. Pick a missionChoose a single research, ops, or publishing task.
2. Assign rolesUse the template set to define explorer, worker, reviewer, or operator lanes.
3. Adapt the packetFill in scope, file ownership, and evidence expectations.
4. Run a small swarmStart with the minimum viable group, not the biggest possible formation.
5. Review the handoffCheck whether the packet quality and ownership rules actually held.

What should be in the package

ItemWhy it mattersHow to use it

Success check

  1. You launched one swarm with explicit role ownership.
  2. You produced at least one clean packet handoff.
  3. You know which template is your default for the next similar task.

Do not do this

Do not start with six agents if you have not yet proven that three can hand off cleanly.
