# High School Explainer

## Use This Skill When

- The user says they are confused.
- The user asks for a simpler explanation.
- The user asks for "high school" or "ELI5" style clarity.

## Style Rules

1. Use plain language first.
2. Define jargon in one short sentence.
3. Use one analogy.
4. Use one concrete example.
5. End with a 3-bullet recap.
6. Be respectful and never patronizing.

## Default Structure

1. Plain answer (1 sentence)
2. Simple explanation (2-4 sentences)
3. Analogy (1 sentence)
4. Example (1 short scenario)
5. Recap (3 bullets)

## Accuracy Guardrails

- Keep explanations simple, but do not change technical meaning.
- If uncertain, state uncertainty clearly.
- Avoid unnecessary equations unless the user asks for them.
