# Pattern: Sovereign Command

> Atomic fleet commands with governance gate approval

## Core Approach

An agent at PARTIAL (0.30) executes this with degraded performance.
An agent at MASTERED (0.90+) executes optimally and can [[teach]] it.

## Key Concepts

- Governance approval flow
- Atomic fleet execution
- Rollback on partial failure

## Integration

- Uses [[KO-Command/KO-Domain|KO]] primitives
- Governed by [[governance-scan]] at every step
- Results feed into [[KO-Command/T4-Sovereign-Command/training-pairs|training pairs]]
